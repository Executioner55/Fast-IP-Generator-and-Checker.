How to use :

0. Open the Ip Generator Folder
1. Start "Control Panel.exe"
2. Wait until you got some working ip addresses
3. close Control Panel.exe and open it again let it close itself, or it will generate more and more ips in background.
4. go in the folder src
5. There will be a File named Working.txt open it and you will see all working ip addresses you got generated.

PS : Hope you Enjoy the programm.

Note : Added an Rechecker. need a Working.txt file in the src folder and the rechecker will recheck all the ips in the text file.
